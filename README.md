# Volt Programming Language Website

The official static website for **Volt**, a strictly typed, schematic-inspired programming language. This repository hosts the landing page: guarantees, syntax that matches the reference interpreter, runnable examples, and how to clone and run [volt-interpreter](https://github.com/ryankkhalil19/volt-interpreter).

🌐 **Live site:** https://ryankkhalil19.github.io/volt-lang-site/

⚙️ **Interpreter source:** https://github.com/ryankkhalil19/volt-interpreter
